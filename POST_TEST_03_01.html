Tiga Cara Menulis CSS

1. Inline CSS

Ini adalah contoh penggunaan inline CSS. CSS ditulis langsung di dalam atribut 'style' pada elemen HTML. Meskipun mudah diterapkan untuk perubahan cepat tetapi cara ini juga memiliki beberapa kekurangan yaitu:
- Sulit untuk dikelola
- Tidak dapat digunakan ulang
- Mencampur konten dengan styling

2. Internal CSS

Ini adalah contoh penggunaan internal CSS. CSS ditulis di dalam tag style di bagian head dokumen HTML. Keuntungannya:
- Tidak perlu file eksternal
- Styles terisolasi untuk satu halaman
- Bisa digunakan ulang dalam satu halaman

3. External CSS

Ini adalah contoh penggunaan external CSS. Biasanya CSS ditulis dalam file terpisah dan di-link ke HTML. Keuntungannya:
- File CSS bisa digunakan untuk banyak halaman
- Pemeliharaan lebih mudah
- File lebih terorganisir
- Cache browser lebih efektif

Cara Menghubungkan External CSS

            <link rel="stylesheet" type="text/css" href="style.css">

Baris kode di atas biasanya ditempatkan di dalam tag <head> untuk menghubungkan file CSS eksternal.
